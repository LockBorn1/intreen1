Postmortem: 2025-09-20 Elevated API errors

Published 2025-09-22

Summary

Misconfigured cache layer caused elevated 5xx rates between 14:12–16:45 UTC.

Impact

~3% of API requests failed for ~2.5 hours.

Root cause

Incorrect TTL on a hot path leading to stale/invalid objects.

Remediations

  • Fix cache TTL policy
  • Improve canary checks
  • Add error budget SLO alert
